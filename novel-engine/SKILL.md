# 📖 单小说生成引擎

## 🎯 核心目标

本系统用于生成**一部可长期连载的网络小说**：

- 支持百万字扩展
- 结构不崩
- 爽点持续输出
- 角色长期成长

---

## ⚙️ 核心能力

### 1️⃣ 世界观构建
→ world_build.md

- 世界结构
- 势力体系
- 力量等级

---

### 2️⃣ 章节生成
→ chapter_write.md

- 单章3000+字
- 四幕结构
- 至少3个爽点

---

### 3️⃣ 钩子系统
→ hook.md

- 每章结尾必须留悬念

---

### 4️⃣ 扩写系统
→ expand.md

- 增强细节
- 去AI味

---

### 5️⃣ 总结系统
→ summary.md

- 每50章滚动记忆

---

## 📥 输入

| 参数       | 说明     |
| ---------- | -------- |
| theme      | 小说主题 |
| chapter_id | 当前章节 |
| summary    | 前情     |

---

## 📤 输出

- 世界观
- 章节正文
- 扩写版本
- 章节总结
