# DeepDigest · 深度信息萃取器

一个面向信息工作者的多模态总结工具。

不仅做摘要，还做：

- 信息提纯
- 结构重组
- 关键洞察提炼
- 可复用输出（适合报告 / 内容创作 / 决策）

---

## ✅ 支持输入

- 🌐 网页 URL
- 📄 PDF / 文档
- 🖼 图片（OCR + 理解）
- 🎧 音频（转录 + 总结）
- 🎬 YouTube 视频

---

## 🎯 输出能力

- 分层摘要（短 / 中 / 长）
- 核心观点提取
- 关键数据抓取
- 逻辑结构重建
- 行动建议（可选）

---

## 🚀 使用方式

```bash
deep_digest "<URL或文件路径>" --mode insight
